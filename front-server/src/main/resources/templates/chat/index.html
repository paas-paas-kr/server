언어와 문화의 장벽 없이 24시간 QnA 챗봇과 대화하세요.
생활, 문화, 행정 등 궁금한 모든 것을 물어보실 수 있습니다.

📖 사용 방법

1
언어 선택
아래 '응답 언어'에서 원하는 언어를 선택하세요. AI가 선택한 언어로 답변합니다.
2
질문 입력
아래 예시 질문을 클릭하거나, 화면 하단 입력창에 자유롭게 질문을 입력하세요.
3
AI 답변 확인
QnA 챗봇이 실시간으로 답변합니다. 추가 질문도 자유롭게 계속할 수 있습니다.
4
음성 파일 업로드 (선택)
입력창 옆 🎤 버튼을 클릭하여 음성 파일을 업로드하면 AI가 음성을 텍스트로 변환하여 답변합니다.
